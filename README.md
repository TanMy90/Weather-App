# Weather-App
Weather Report is a Desktop Application that displays real time weather data of the city which is given as input.

Weather Report is a Desktop Application that displays real time weather data of the city which is given as input. This application uses the Open Weather Map API to request real time data.The API uses RESTful calls issued in JSON format.The application displays last updated, weather, Temperature, Wind, Humidity, visibility, sunrise, sunset data.

To Run

Python weather.py
